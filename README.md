# firstrepo

Data science and Python learning sandbox — notebooks, experiments, and notes.

> Originally my first GitHub repo (2020). Revived in 2026 as an active learning space.

## Structure

```
firstrepo/
├── README.md
├── notebooks/
│   └── hello_python.ipynb
└── scripts/
    └── hello.py
```

## Quick start

```bash
python -m venv .venv
.venv\Scripts\activate
pip install jupyter pandas matplotlib
jupyter notebook notebooks/hello_python.ipynb
```

Or run the simple script:

```bash
python scripts/hello.py
```

## Topics

- Python basics
- Jupyter notebooks
- Data exploration (pandas, matplotlib)
